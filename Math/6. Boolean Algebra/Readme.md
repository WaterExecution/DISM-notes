# Distributive Laws
```
x•(y+z) = (x•y) + (x•z)
x+(y•z) = (x+y) • (x+y)
```
# Identity Laws
```
x•1=x
x+0=x
```
# Complement Laws
```
x•x̄=0
x+x̄=1
```
# Idempotent Laws
```
x•x=x
x+x=x
```
# Bound Laws
```
x•0=0
x+1=1
```
