  • Infocomm Security
  • Linux Administration and Security
  • Security Policy and Incident Management

 Security Analyst
  • Infocomm Security
  • Ethical Hacking and Defences

 Security Software Developer
  • Secure coding

  • Security Policy and Incident Management
 Develop and communicate risk policies for an organization

  • Security Policy and Incident Management
 Investigate and document cybersecurity incidents within an organization
